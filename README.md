# credit-card-fraud-detection
Built a fraud detection model using Random Forest for feature selection and Logistic Regression for classification. Applied data preprocessing with StandardScaler and handled class imbalance using SMOTE. Evaluated performance with Accuracy, Precision, Recall, F1-Score, and AUC-ROC
The dataset used is https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
